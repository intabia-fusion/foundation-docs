Foundation — это платформа управления проектами «всё-в-одном» с открытым исходным кодом, предлагающая альтернативу таким инструментам, как Linear, Jira, Slack, Notion и Motion.

Foundation является форком проекта <a href="https://github.com/hcengineering/platform" target="_blank">hcengineering/platform</a>, основываясь на его надёжном фундаменте для создания улучшенного решения для совместной работы и управления проектами.

Предоставляя мощный набор функций, включая управление рабочими процессами, личное планирование, совместное редактирование и видеоконференции, Foundation оптимизирует процессы управления проектами таким образом, который органично согласуется со сложностью и динамичной природой реальных рабочих процессов.

Foundation Docs содержит информацию обо всех основных функциях платформы Foundation. Вы также можете узнать больше о Foundation в моем <a href="https://haiodo.github.io" target="_blank">блоге</a>.

## Куда мы движемся

По мере того как мы фокусируемся на наших целях для будущего платформы Foundation, мы концентрируемся на создании лучшей системы управления процессами, которая действительно отражает реальные, динамичные рабочие процессы. Мы перерабатываем всю структуру платформы Foundation не только для поддержки этой цели, но и для закладки фундамента нашего видения глобального Foundation, который станет единственным инструментом, который когда-либо понадобится кому-либо для управления своей работой.
