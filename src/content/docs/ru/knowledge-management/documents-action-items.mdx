Иногда вам может понадобиться создать активности для членов команды непосредственно в документе, не привязывая их к какой-либо конкретной задаче в Трекере. Это может быть полезно для небольших задач, которые не требуют полноценной задачи, напоминаний или других дел, которые нужно выполнить и которые не связаны с конкретным проектом или задачей.

## Создание активностей

Есть несколько способов создать активности непосредственно в документах:
- **Сочетание клавиш** — введите `[]`, затем `пробел` на новой строке. Появится пустой чекбокс.
- **Плавающая панель инструментов** — наведите курсор на текст и нажмите на значок `+`, который появится слева. Выберите `Задача`, чтобы создать новую активность.
- **Слеш-команда** — введите `/`, чтобы открыть выпадающее меню опций. Выберите значок чекбокса, чтобы создать новую активность.

## Назначение активностей

После создания активности любым из вышеперечисленных способов вы можете назначить её члену команды. Обратите внимание на серый значок профиля, который появляется слева от чекбокса. Нажмите на этот значок, чтобы выбрать члена команды для назначения задачи.

Когда активность назначается члену команды в документе, этот человек получит уведомление о том, что ему назначена активность, и он сможет запланировать задачу в своём Планировщике.
