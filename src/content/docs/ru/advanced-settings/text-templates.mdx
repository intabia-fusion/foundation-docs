## Что такое текстовые шаблоны?

Текстовые шаблоны могут помочь вам сэкономить время и облегчить создание задач. С помощью текстовых шаблонов вы можете создавать предопределённые текстовые структуры, которые можно использовать для описаний задач, чатов, комментариев и многого другого.

Текстовые шаблоны позволяют определять многоразовые структуры контента, которые можно использовать в различных функциях вашего рабочего пространства. Эти шаблоны помогают обеспечить согласованность, уменьшить повторяющийся ввод текста и улучшить совместную работу, предоставляя чёткие, стандартизированные форматы.

> **Примечание:** Опция `Поле` ещё находится в разработке, поэтому рекомендуется вместо этого создавать собственные «поля» в вашем шаблоне, которые нужно заполнять при использовании.

## Создание и управление текстовыми шаблонами

Для создания и управления текстовыми шаблонами перейдите в Настройки и выберите `Текстовые шаблоны` в области `РАБОЧЕЕ ПРОСТРАНСТВО` в левом боковом меню. Здесь вы можете создавать новые шаблоны или редактировать существующие.

Чтобы создать новый текстовый шаблон, нажмите кнопку `Создать шаблон`. Введите название для шаблона, определите структуру, форматирование и заполнители, и сохраните шаблон в папку.

По умолчанию новые шаблоны сохраняются в папку `Публичные шаблоны`, где они доступны для всех в рабочем пространстве. Чтобы сделать ваши шаблоны приватными, создайте новую группу шаблонов и переключите настройку конфиденциальности.

## Использование текстовых шаблонов

После создания текстовые шаблоны можно получить из сообщений чата, комментариев, описаний задач и в любом другом месте, где доступен значок текстового шаблона. Просто нажмите на значок текстового шаблона и выберите шаблон, который хотите использовать.

Есть много областей вашего рабочего пространства, где текстовые шаблоны полезны. Вот несколько примеров вариантов использования:

- **Повторяющиеся описания задач:** Определите конкретное форматирование для повторяющихся задач или просто создайте базовое многоразовое оформление для вашего контента.

- **Быстрые ответы в чате**: Используйте заготовленные ответы для часто задаваемых вопросов или обновлений.

- **Приглашения на встречи**: Стандартизируйте объявления о повторяющихся встречах:

**_Пример:_**
```
    Напоминание: Наша [Название встречи] запланирована на [Дата и время].
    Пожалуйста, ознакомьтесь с повесткой дня здесь: [Ссылка].
```
- **Ревью кода**: Используйте шаблоны для стандартизации обратной связи по коду:

**_Пример:_**
```
    Код: [Ссылка на файл/строку]
    Наблюдение: [Проблема/Обратная связь]
    Предлагаемое изменение: [Предложенное решение]
```
