## Навигация по командному планировщику

Командный планировщик в "Интабия. Платформа" представляет из себя способ визуализации расписания вашей команды. Чтобы перейти к командному планировщику, нажмите на значок солнца в левой панели навигации.

Командный планировщик разбивает расписание по проектам, и вы можете найти каждый из ваших проектов в левой боковой панели.

Это позволяет командам с несколькими проектами легко видеть всю запланированную работу для каждого проекта. Это также обеспечивает конфиденциальность проекта, поскольку члены команды смогут просматривать планировщик только для проектов, участниками которых они являются, в то время как владелец рабочего пространства сможет просматривать все проекты. По этой причине в настоящее время невозможно просматривать работу для всех проектов одновременно.

### Представления расписания и календаря

По умолчанию командный планировщик показывает представление расписания предыдущего и текущего дня. Вы можете прокручивать дни с помощью кнопок со стрелками в правом верхнем углу, просматривая по 2 дня за раз при прокрутке.

Представление календаря отображает либо ежедневный, либо еженедельный календарь задач каждого члена команды. В ежедневном календаре задачи отображаются в виде блоков и могут быть просмотрены с шагом в 1 час или 30 минут. Еженедельный календарь отображает задачи каждого члена команды на каждый день недели.

## Настройки видимости для активностей

По умолчанию, когда член команды планирует активность в своём личном планировщике (см. [Планирование активностей](/ru/task-tracking/scheduling-action-items)), она автоматически станет видимой в командном планировщике для всех остальных членов команды. Название активности, связанная задача и количество запланированных часов — всё это оторажено в календаре. Нажатие на стрелку раскрывающегося списка рядом с этой задачей также покажет конкретный временной блок, на который запланирована активность.

Если член команды установил видимость своей активности как «Видно только вам», она не будет отображаться в командном планировщике.

Активности, не связанные с пространством (например, те, которые созданы вручную в личном планировщике человека), не будут отображаться в командном планировщике. После добавления активности в личный планировщик пользователь может выбрать настройку видимости для этой активности в зависимости от того, хочет ли он, чтобы она была видна всем, отображалась как «Занят» или была видна только ему.

## Настройки видимости для событий

События в личном планировщике члена команды также будут отображаться в командном планировщике в зависимости от настроек видимости.

Если видимость установлена как «Видно всем», каждый в команде сможет видеть название, время и продолжительность события. Если видимость установлена как «Занят», члены команды смогут видеть, что человек занят в это время, но они не смогут видеть название события.

Как и в случае с активностями, если видимость установлена как «Видно только вам», событие не будет отображаться в командном планировщике.
