Модуль Products (включён по умолчанию в рабочих пространствах [TraceX](/foundation-docs/ru/getting-started/introduction-tracex)) используется для управления версиями продуктов и документацией. Продукты могут представлять физические или цифровые изделия вашей организации и могут быть связаны с другими объектами в рабочей области.

## Создание продукта

Для создания продукта нажмите кнопку `+ Product` в модуле Products.

Укажите название и описание, выберите участников и роли по необходимости, затем нажмите `Create`.

## Creating a product version

> **Примечание:** Чтобы создать новую версию продукта, вам потребуется действующий документ Change Control, оформленный по шаблону Change Control. Подробнее об этом шаге см. руководство по созданию [шаблонов](/foundation-docs/ru/controlled-documents/templates).

Чтобы создать новую версию продукта в модуле Products, откройте страницу сведений о продукте, для которого хотите добавить версию. Затем нажмите кнопку `+` рядом с разделом `Product Versions`.

Далее вы сможете указать название и описание новой версии и выбрать, является ли изменение существенным (major) или несущественным (minor). Также можно выбрать документ Change Control из выпадающего списка.

После заполнения необходимых сведений нажмите `Create`. Новая версия будет добавлена в историю версий продукта.
