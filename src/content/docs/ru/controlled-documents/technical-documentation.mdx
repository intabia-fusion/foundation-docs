Техническая документация — это раздел модуля «Контролируемые документы», предназначенный для управления записями, связанными с разработкой продукта, такими как отчёты о тестировании и документы с требованиями.

## Создание технического документа

Для создания нового технического документа откройте меню настроек рядом с любым пространством продукта в подменю `TECHNICAL DOCUMENTATION` в левой боковой панели и выберите `Create new document`. Также можно нажать кнопку `+ New document` в модуле «Controlled Documents».

> **Примечание:** Новый технический документ также можно создать как дочерний элемент любого существующего документа.

Выполните следующие шаги для создания документа:

**Шаг 1:** Выберите пространство и родительский объект для документа. Пространство определяет применяемые роли и права доступа; родитель указывает, где документ будет храниться. Документ может быть непосредственным дочерним элементом пространства (верхний уровень) или дочерним элементом другого документа внутри пространства.
