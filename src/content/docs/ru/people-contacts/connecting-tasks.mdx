---
title: Связывание задач
description: Узнайте, как связывать задачи с вашими контактами.
---

Задачи в вашем рабочем пространстве могут быть напрямую связаны с людьми в вашем списке контактов. Есть несколько случаев, когда это может быть полезно, например:

- **Работа с клиентами** — фрилансеры, агентства и компании, использующие "Интабия. Платформа" для управления клиентскими проектами, могут связывать задачи в своём Трекере с контактами клиентов. Это позволяет связать задачи с контактной информацией клиента, что упрощает отслеживание того, с кем связаться для получения обновлений, обратной связи или утверждений.

- **Планирование последующих контактов** — отслеживайте последующие разговоры, создавая задачи в вашем Трекере, связанные с вашими внешними контактами. Ведите записи о ваших взаимодействиях и устанавливайте напоминания о том, когда связаться повторно.

- **Отслеживание запросов поддержки** — если вы используете "Иньабия. Платформа" для управления запросами поддержки, вы можете связывать задачи с контактом, который отправил запрос. Это позволяет легко видеть все задачи, связанные с конкретным контактом, и отслеживать статус их запросов.

## Создание новой задачи

Чтобы создать задачу из контакта, кликните на него правой кнопокой мыши и выберите `Звести связанную задачу`. Откроется диалоговое окно для создания новой задачи.

После создания задачи в боковой панели деталей появится поле `Связан`. Здесь вы можете увидеть контакт, связанный с этой задачей.

## Связывание существующей задачи

Чтобы связать существующую задачу с контактом, откройте задачу и откройте выпадающее меню настроек. Выберите `Зависимости`, затем `Связать с задачей`. Отсюда вы можете выбрать контакт для связи с задачей.

> **Примечание:** Здесь вы также можете связывать другие задачи, документы и объекты

## Просмотр связанных задач

Чтобы просмотреть все задачи, связанные с контактом, перейдите в Трекер и выберите `Все задачи`. Используйте кнопку `Фильтр`, чтобы выбрать `Связано`, а затем выберите имя контакта, задачи которого хотите увидеть.
