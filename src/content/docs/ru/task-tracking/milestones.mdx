## Что такое этап?

Этапы — это способ организации задач в "Интабия. Платформа". Они могут использоваться для группировки задач на основе общей темы, проекта или временной шкалы. Например, вы можете создать этап для конкретного релиза функции, спринта или фазы проекта. Этапы можно использовать для отслеживания прогресса, установки дедлайнов и обеспечения того, чтобы задачи выполнялись в логическом порядке.

> **Примечание:** Этапы специфичны для проекта и не могут быть перемещены в другой проект в Трекере.

## Создание нового этапа

Перейдите в Трекер и выберите Этапы под любым проектом. Затем нажмите кнопку `+ Этап` в правом верхнем углу. Здесь вы можете ввести название и необязательное описание для вашего Этапа, а также установить статус и дедлайн.

## Добавление задач к этапу

### Создание новой задачи внутри этапа

Есть несколько способов добавить задачи к этапу. Если вы ещё не создали задачу, просто нажмите кнопку `+` на странице деталей любого этапа. Это позволит вам создать новую задачу, которая будет автоматически связана с этим этапом.

В качестве альтернативы, при создании новой задачи из любого другого места в трекере, просто выберите желаемый этап из выпадающего меню в диалоговом окне создания задачи:

## Использование этапа для сортировки задач

Как и любой другой атрибут задачи, этап можно использовать для фильтрации и сортировки задач в Трекере. Например, вы можете переключать, хотите ли вы видеть этап для каждой задачи в вашем Трекере, или вы можете настроить весь вид Трекера, организовав задачи по этапам.

Задачи также можно фильтровать по этапам, что позволяет быстро увидеть все задачи, связанные с выбранным этапом.
