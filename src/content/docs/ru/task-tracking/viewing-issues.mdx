## Представления списка и канбан

Представление по умолчанию для задач — это представление списка, где задачи перечислены в порядке их статуса. Foundation также предлагает возможность канбан-представления, которое отображает задачи в формате столбцов. Чтобы переключаться между двумя представлениями, нажмите на значок списка или канбана в правом верхнем углу окна.

При просмотре задач для конкретного проекта вы сможете просматривать только один [тип задач](/ru/advanced-settings/task-types) за раз. Это связано с тем, что типы задач могут иметь разные состояния процесса. Чтобы просмотреть все ваши задачи в одном месте (включая все типы задач), перейдите к `Все задачи` в боковой панели.

## Быстрый просмотр задачи

Нажмите клавишу `пробел`, наведя курсор на любую задачу, чтобы «заглянуть» в детали этой задачи. В этом представлении вы можете использовать клавиши вверх и вниз на клавиатуре для прокрутки ваших задач.

Наведение курсора на комментарии или сроки выполнения также откроет предпросмотр этих деталей.

## Сочетание клавиш

Нажмите `C` на клавиатуре, чтобы создать новую задачу из любого представления.

## Сортировка, фильтрация и поиск

Опции сортировки и фильтрации задач также находятся в правом верхнем углу. Кнопка `Вид` позволяет настроить некоторые параметры просмотра, а также опции сортировки по группировке и порядку задач на странице. Кнопка `Показать` отображает переключатели для показа и скрытия деталей задач.

Кнопка `Фильтр`, рядом с панелью поиска, предлагает опции для фильтрации задач. Панель поиска можно использовать для поиска конкретной задачи по ключевому слову.

Для быстрой сортировки выберите `Активные`, `Бэклог` или `Все` из меню над панелью поиска.
