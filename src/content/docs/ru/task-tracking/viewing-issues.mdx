## Представления списком и канбан

Представление задач по умолчанию — это представление задач списком, где они перечислены в порядке их статуса. В "Интабия. Платформа" также есть возможность канбан-представления, которое отображает задачи в формате плашек и столбцов. Чтобы переключаться между двумя представлениями, нажмите на значок списка или канбана в левом верхнем углу окна над списком задач.

При просмотре задач для конкретного проекта вы сможете просматривать только один [тип задач](/ru/advanced-settings/task-types) за раз. Это связано с тем, что типы задач могут иметь разные состояния процесса. Чтобы просмотреть все ваши задачи в одном месте (включая все типы задач), перейдите к `Все задачи` в боковой панели.

## Быстрый просмотр задачи

Нажмите клавишу `пробел`, наведя курсор на любую задачу, чтобы открыть детали этой задачи. При открытых деталях задачи вы сможете использовать клавиши вверх и вниз на клавиатуре для прокрутки ваших задач. Повторное нажатие на пробел закроет окно информации о задаче.

Наведение курсора на комментарии или сроки выполнения также откроет предпросмотр этих деталей.

## Сочетание клавиш

Нажмите `C` на клавиатуре, чтобы создать новую задачу из любого представления Трекера.

## Сортировка, фильтрация и поиск

Опции сортировки и фильтрации задач также находятся в правом верхнем углу. Кнопка `Группировка и сортировка` позволяет настроить некоторые параметры просмотра, а также опции сортировки по группировке и порядку задач на странице. Кнопка `Настроить отображние` отображает переключатели для показа и скрытия деталей задач.

Кнопка `Фильтр`, рядом с панелью поиска, открыавает опции для фильтрации задач. Панель поиска можно использовать для поиска конкретной задачи по ключевому слову.

Для быстрой сортировки выберите `Активные`, `Пул задач` или `Все` из меню над панелью поиска.
